Exercício de HTML desenvolvido em aula na faculdade Flamingo. O projeto apresenta uma página simples de receita de bolo, usada para praticar estrutura básica de uma página web.

## Conteúdo

- Arquivo HTML da atividade.
- Estrutura com texto, títulos e conteúdo de receita.
- Prática inicial de marcação HTML.

## Tecnologias

- HTML5
- Git e GitHub
