# Receita-de-bolo

Exercicio de HTML desenvolvido em aula na faculdade Flamingo. O projeto apresenta uma pagina simples de receita de bolo, usada para praticar estrutura basica de uma pagina web.

## Conteudo

- Arquivo HTML da atividade.
- Estrutura com texto, titulos e conteudo de receita.
- Pratica inicial de marcacao HTML.

## Tecnologias

- HTML5
- Git e GitHub

## Como visualizar

Acesse pelo GitHub Pages:

https://matheusgdqueiroz-del.github.io/Receita-de-bolo/

Ou baixe o repositorio e abra o arquivo `.html` no navegador.
